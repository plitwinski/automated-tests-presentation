# How to run examples

## prerequisites

### example 01, 02, 04, 05
.NET Core 2.0 SDK (or higher)

### example 03
nodejs  6.9.5 (or higher)

## example 01
#### run following commands
* dotnet restore
* cd Example.CoarseGrainedUTest.Tests 
* dotnet test
#### or
#### open, compile and run via test explorer in Visual Studio 2017
